# QuickSOS

## :globe_with_meridians: English version :globe_with_meridians:

## üìù About the application

QuickSOS is an Android mobile application developed as a final project, designed to unify in one place the main emergency and information phone numbers at the national level. Although the number 112 is widely known, users often need to contact specific services (such as medical information, veterinary care, or their assigned health center), whose location and dialing may be slow in stressful or urgent situations.

The application has been developed using Android as the main platform, with Firebase as the backend for data management. It is intended for any user with an Android device and an internet connection. Among its main features are direct dialing of emergency numbers from within the app and the ability to add custom contacts of personal interest for each user.

Regarding language support, it offers translations in Spanish, Galician, Basque, Catalan, English, Portuguese, French, German, and Italian. In the long term, emergency numbers from other countries could also be added, opening the possibility for international use.

## :eyes: How to test the project

After downloading the repository and opening it with your IDE, there are two ways to test the application:
 - `Login as guest`: The fastest and easiest option, but only the default contacts list can be viewed.
 - `Register a new account`: You can register a test account (which can later be deleted), allowing you to add personal contacts and change the user password.
